Pokémon Duke Blue


Pokémon Duke Blue is a role-playing videogame built with JavaFX that is based off of the popular Pokémon videogame series. I built the game because I wanted to learn JavaFX and more about graphical user interfaces.

The game features a navigable world that the player explores by controlling a sprite. The player can interact with several non-player-characters with the sprite and can also challenge the game boss to a turn-style Pokémon battle. Sound effects and cheat codes are also included.

You can download and play the game yourself here. Export the ZIP archive into your file system and double-click the JAR file to start the game. The program's repository along with complete gameplay instructions are available here.

CellSociety


CellSociety is a cellular automata simulation engine coded in Java using the JavaFX framework. I developed this application with two other team members. I primarily served as the project's front-end developer, overseeing GUI development, CSS styling, and simulation control. I was additionally in charge of interfacing the front-end GUI with the program's back-end logic that powers each simulation.

The application simulates several different cellular automata simulations along with GUI controls that allow the user to customize how each simulation is executed. The software stack also includes a DOM parser for XML; this allowed for further modifications of simulation parameters through XML files tailored for each type of simulation.

You can download and run the simulations yourself here. Export the ZIP archive into your file system and double-click the JAR file to start the application. The program's repository along with complete instructions are available here.
